# Docker 4 Laravel

```
These files have to be downloded and placed in the root folder of your Laravel application so at the level of your app folder.
```


## Services Versions

This docker will work for Laravel 5.5 and latest.

- FROM nginx:latest
- FROM php:7.1-fpm
- FROM mysql:5.7
